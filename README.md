# CRM Dashboard - Textek Analyst

Interactive Customer Relationship Management (CRM) dashboard built using **Streamlit** and **Excel**.  
This project allows you to analyze customer behavior, categorize buyers, track growth, and visualize product popularity.

## Features

1. **VIP Buyer**
   - Add, delete, and rank top customers by total transactions.
   - Highlights top 10 buyers with color coding.

2. **Kategori Buyer**
   - Classify customers based on activity status.
   - Displays counts and average purchase intervals per category.

3. **Marketing Ads**
   - Pie chart visualization of marketing channel effectiveness.
   - Highlights the most successful channel.

4. **Pertumbuhan Pelanggan**
   - Track customer growth over months.
   - Add and remove monthly data.
   - Trend indicators with color coding.

5. **Produk Populer**
   - List products sorted by purchase count.
   - Highlights top 5 products.

6. **Produk Favorit Customer**
   - Shows customer favorite products with purchase counts.

## Tech Stack

- Python
- Streamlit
- Pandas
- Openpyxl
- Matplotlib (for plots)
- Excel as data source
